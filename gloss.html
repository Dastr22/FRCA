Fast Response Cache Accelerator (FRCA)

The Fast Response Cache Accelerator (FRCA) is a feature of the HTTP Server that improves performance by caching static files such as text and images. It serves cached files from the TCP/IP layer, bypassing the HTTP Server for faster response times. FRCA operates only when the connection is not secure (non-SSL) and does not cache dynamic content or protected pages.

FRCA automatically caches eligible files without requiring explicit listing, but sensitive files should be excluded using the FRCANoCachin directive to ensure proper authentication.
