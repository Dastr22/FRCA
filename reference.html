FRCA Directives

  • EnableFRCA

Enables or disables the Fast Response Cache Accelerator.

  • Syntax: EnableFRCA On | Off
  • Default: Off
  • Example:
    EnableFRCA On
  • FRCANoCaching

Specifies which URIs should not be cached by FRCA.

  • Syntax: FRCANoCaching /path/to/file
  • Default: Not set
  • Example:
    FRCANoCaching /usergroup/*.html
FRCANoCaching /Docs/*.pdf
